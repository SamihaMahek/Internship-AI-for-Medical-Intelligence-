# Internship-Ai-for-medical-intelligence
This repository contains the core programmatic tasks, logic workflows, and system utilities developed during the AI for Medical Intelligence Internship program.

---

## 🚀 Included Tasks & Core Logic

### 1. 🔒 Secure Password Verification (`checkpassword.py`)
A security utility simulating access gateway controls for medical record database systems.
* **Core Logic:** Implements an automated lookup loop to validate access tokens.
* **Security Control:** Includes a 3-attempt safety cutoff matrix to simulate network security lockouts.

### 2. 📋 Clinical Duty Tracker (`attendancemonitor.py`)
A database helper utility built to log and organize intern shifts and on-call rotations.
* **Core Logic:** Dynamically reads user metrics and assigns statuses using a standard relational data map.
* **Output Format:** Generates a structured terminal summary view of active shift configurations.

### 3. 🤖 Patient Care Assistant Bot (`minimoodchat.py`)
An interactive automated responder designed to simulate automated conversational care support channels.
* **Core Logic:** Uses keyword classification rules to parse patient inputs.
* **Safety Rules:** Delivers tailored clinical baseline recommendations based on recognized symptoms (e.g., fatigue or physical discomfort).

---

## 🛠️ Technology Stack
* **Programming Language:** Python 3.10+
* **Execution Environment:** Local Terminal / Command Line Interface
